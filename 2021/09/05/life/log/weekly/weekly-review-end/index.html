《每周回顾终》

每周回顾

2021-10-31

本周总结

  • 上传图片策略优化
    • 上传时同时上传origin与mini
    • 前端展示为mini,不用担心盗用origin,必须要加签才可以访问
    • 上传ui
    • 拖动移动优化
    • 拖动删除
    • 重新上传
    • 上传cos与发布到后端连调
    • 删除

总结

  • 勇者斗恶龙通关
  • 和胜楠帝国时代4

下周目标

  • 这次肯定可以完成了,后续没有困难点了,只剩优化了
  • 完成集邮的发布,因为发布流程与coer发布流程类似,坑都踩过了

2021-10-22

本周总结

  • 本周主要研究腾讯的cos
    • 前端通过objectName生成加签路径访问图片(存在限时),前端通过objectName作为key缓存图片
    • 前端通过接口获取临时密钥,然后在前端加签,生成临时签名
    • 前端上传图片时携带临时签名,cos返回objectName,前端验证objectName是否缓存,未缓存加签objectName生成加签路径

总结

  • 胜楠打扫厨房
  • 李子晗清扫公厕,脏的心态爆炸
  • 胜楠下厨了两天的可乐鸡翅,我回了一次蒸虾
  • 大黑要提前给我房租,估计是上个月警告他下次及时交房租不开心了,耍脾气,那我能惯着他吗

下周目标

  • 本周因为完成coser照片的发布

2021-10-18

本周总结

  • 本周主要围绕相册上传
  • 将储存策略由自主搭建平台改为云平台,比较省心,稳定性有保障,除了贵没什么坏处
  • 发布前预览策略70%,差上传照片后的插入

总结

  • 恢复运动
  • 显示器售后
  • 周一酱骨-周五烤肉-周日烧烤
  • 14日清理主厕卫生,心态很爆炸
  • 17日胜楠将厨房卫生打扫的非常干净

下周目标

  • 完成coser照片的发布
  • 把上传策略优化并且钉死

2021-10-10

本周总结

  • 本周主要是在优化request结构,并将以往的request修改,将代码简化了95%,真正意义上的复制粘贴
  • 修复photoContent的一些bug,比如退出页面再回来top会有一部分空白填充,会将箭头顶出来
  • 进一步理解EnvironmentObject,link页面1->back->link页面1,重复此过程,最后link页面1可能会有n次的request结果,我忽略了一个问题,
    因为request结果是进入environment的,所以他会在内存中一直保留 当初决定全部封成environment是因为觉得我在其他页面3可能修改页面1属性,觉得方便就这样做了 在理解到StateObject与environemntObject只是不同的声明方式时,我的结构没有改变,那我感觉绑定页面model好了
    代码结构不变,效果优化,nice

总结

  • 和胜楠有一点不愉快
  • 胜楠要被迫辞职,局面不乐观
  • 游戏get dq11、任天堂大乱斗、马里奥网球
  • 食谱get 蒸牛肉
  • 尝试开始恢复运动

下周目标

  • 完成coser照片的发布
  • 完善现有的photo模块

2021-09-30

本周总结

  • 未完成目标,fastdfs用docker搭建有些坑要踩,花了3天时间,后面就因为种种原因被打断,这个问题我会解决,主要还是对前端的抗拒导致的
  • 完成部分:后端文件上传,前端发布照片的页面进行了调整,正在对接
  • 和胜楠过山车,和烨董恰饭吐槽,去了猫咖,完美的生活在舒适圈,一定要脱离!

    总结

  • 10月了,最后期限3个月,前端再讨厌也要尽快完结

下周目标

  • 完成coser照片的发布
  • 完善现有的photo模块

2021-09-22

本周总结

七日

  • b站视频播放量突然猛增到1.5w,小程序随着b站的播放量同样猛增一段,并且收获5次(50元)打赏
  • 维护了几天小程序,接下来怠惰了好多好多好多天

    总结

  • 怠惰挂机,反思了一下,实在有点搞不动swiftui的苹果开发,有各种不兼容不合适的地方,无法满足我心中的交互效果,
    对自己的无能感到憎恨.最开始应该选择uikit比较靠谱,但是uikit上手难度又比swiftui难度再高一些,或者说写起来没有swiftui爽
  • 在经过小程序的鼓舞后,更加想尽快将app做出来,重新振作一下
  • 现在剩余模块为
    • 相集(上传照片模块)
    • 评论模块
    • 我的模块
    • 用户中心模块
  • 原型修改与添加
    • 支付与提现模块

下周目标

  • 完成coser照片的发布
  • 完善现有的photo模块

2021-09-05

本周总结

七日

  • b站视频意外的多了起来,虽然还是很少,点赞收藏基本到20+了,有意思有意思
  • 沉迷两日半的异度之刃2
  • 本周工作主要解决
    1. 加载网络图片并缓存(完成)
    2. 图像查看器(完成)
    3. 图像查看器的缩放动画效果(失败,改为边框)
    4. 解决navigationView与tabview兼容问题,但是未来可能出现其他问题
  • 未来的难点:
    1. 图片选择
    2. 图片压缩
    3. userInfoCenter的UI

      总结

  • 连续搞了10天突然有点萎靡,周五、周六、周日沉迷游戏

下周目标

  • 完善早季客户端,尽可能做完
  • 尝试五笔
